CASwitch类说明

基类
CAControl
头文件
CASwitch.h

概述

CASwitch类用来创建和管理ON/OFF按钮。

CASwitch类声明了一些属性和方法来控制ON／OFF状态。当CASwitch状态发生变化时，会发送CAControlTouchUpInSide事件。

可以通过设置CAImage来自定义CASwitch控件的外观。

功能

创建和初始化CASwitch

  • createWithFrame
  • initWithFrame

设置ON/OFF的状态

  • setIsOn

自定义CASwitch外观

  • setOnImage
  • setOffImage
  • setThumbTintImage

类方法

  • createWithFrame

    返回一个CASwitch对象。

    static CASwitch* createWithFrame(const CCRect& rect);

    返回值

    返回一个CASwitch对象。

    参数

    rect
    新建CASwitch对象的frame。

  • initWithFrame

    初始化CASwitch对象的frame。

    bool initWithFrame(const CCRect& rect);

    返回值

    成功返回true，失败返回false。

    参数

    rect
    初始化CASwitch对象的frame。

  • setIsOn

    设置CASwitch对象的开关状态。

    void setIsOn(bool on);

    返回值

    无

    参数

    on
    true表示开，false表示关。

  • setOnImage

    设置CASwitch对象的ON状态显示的图像。

    void setOnImage(CAImage* onImage);

    返回值

    无

    参数

    onImage
    ON状态下的image。

  • setOffImage

    设置CASwitch对象的OFF状态显示的图像。

    void setOffImage(CAImage* offImage);

    返回值

    无

    参数

    offImage
    OFF状态下的image。
