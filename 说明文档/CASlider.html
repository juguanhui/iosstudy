CASlider类说明

基类
CAControl
头文件
CASlider.h

概述

CASlider类用来在一段连续的取值范围内设定一个值。比如音乐音量设置中会使用到。

可以通过设置CAImage来自定义CASlider控件的外观。

功能

创建和初始化CASlider

  • createWithFrame
  • initWithFrame

设置slider取值

  • setValue
  • setMinValue
  • setMaxValue

自定义CASlider外观

  • setTrackHeight
  • setMinTrackTintImage
  • setMaxTrackTintImage

类方法

  • createWithFrame

    返回一个CASlider对象。

    static CASlider* createWithFrame(const CCRect& rect);

    返回值

    返回一个CASlider对象。

    参数

    rect
    新建CASlider对象的frame。

  • initWithFrame

    初始化CASlider对象的frame。

    bool initWithFrame(const CCRect& rect);

    返回值

    成功返回true，失败返回false。

    参数

    rect
    初始化CASlider对象的frame。

  • setValue

    设置CASlider对象的值。

    void setValue(float value)

    返回值

    无

    参数

    value
    value必须在minValue和maxValue之间。

  • setMinValue

    设置CASlider最小取值。

    void setMinValue(float minValue)

    返回值

    无

    参数

    minValue
    CASlider的最小取值。

  • setMaxValue

    设置CASlider最大值。

    void setMaxValue(float maxValue)

    返回值

    无

    参数

    maxValue
    CASlider的最大值。

  • setMinTrackTintImage

    设置CASlider对象小值区域的图像。

    void setMinTrackTintImage(CAImage* minTrackTintImage);

    返回值

    无

    参数

    minTrackTintImage
    CASlider对象小值区域的图像。

  • setMaxTrackTintImage

    设置CASlider对象大值区域的图像。

    void setMaxTrackTintImage(CAImage* maxTrackTintImage);

    返回值

    无

    参数

    maxTrackTintImage
    CASlider对象大值区域的图像。
