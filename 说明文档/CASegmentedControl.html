CASegmentedControl类说明

基类
CAControl
头文件
CASegmentedControl.h

概述

CASegmentedControl对象是由多个CAButton组成的水平方向的控件。

每一个单独的CAButton可以显示一个标题或者一个图像。CASegmentedControl会自动调整每个CAButton的大小。

类方法

  • createWithFrame

    返回一个CASegmentedControl对象。

    static CASegmentedControl* createWithFrame(const CCRect& rect);

    返回值

    返回一个CASegmentedControl对象。

    参数

    rect
    新建CASegmentedControl对象的frame。

  • initWithFrame

    初始化CASegmentedControl对象的frame。

    bool initWithFrame(const CCRect& rect);

    返回值

    成功返回true，失败返回false。

    参数

    rect
    初始化CASegmentedControl对象的frame。

  • insertSegmentWithTitle

    通过给定的title创建一个segment，插入在CASegmentedControl指定的位置。

    virtual bool insertSegmentWithTitle(const char* title, int index, CAControlState controlState);

    返回值

    操作成功返回true，失败返回false。

    参数

    title
    CAButton在controlState下显示的标题。
    index
    index的区间在[0，getItemCount())之间时，插入在index的位置；index < 0, 插入为第一个元素；index >= getItemCount()，插入为最后一个元素。
    controlState
    CAButton标题显示的状态。

  • insertSegmentWithBackgroundImage

    参数

    CAButton在controlState下显示的背景图片。
    index
    index的区间在[0，getItemCount())之间时，插入在index的位置；index < 0, 插入为第一个元素；index >= getItemCount()，插入为最后一个元素。
    controlState
    CAButton背景图片显示的状态。
